Timestamp Microservice

A simple API to parse natural dates and Unix timestamps.

Example Usage

Try making a GET request to these endpoints:

  • Date String [project_url]/api/2015-12-25
  • Unix Timestamp [project_url]/api/1451001600000

Example Output

{ "unix": 1451001600000, "utc": "Fri, 25 Dec 2015 00:00:00 GMT" }
